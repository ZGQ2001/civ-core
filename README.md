# Civil-Auto-Workspace (CAW) v0.2.3-dev

## 项目简介

Civil-Auto-Workspace 是为土木工程检测与报告编制场景设计的自动化辅助工具集，聚焦于工程师的内业报告编写效率与格式规范性。

本仓库当前处于架构升级阶段：

- 主要工作为从旧版程序迁移到新架构，并对现有业务逻辑进行模块化、分层和配置规范化。
- 新架构信息与迁移原则已经记录在 `00_使用手册/检测辅助工具开发与重构原则总纲.md` 中。

## 当前状态

- `src/civil_auto/`：新架构下的源码目录，已开始逐步重构并补齐目标目录结构。
- `02_Core/`：旧版本程序所在目录，当前仍可直接使用旧程序功能。

## 说明与使用建议

- 若需要了解项目架构与开发原则，请参阅：
  - `00_使用手册/检测辅助工具开发与重构原则总纲.md`
- 若需要运行旧程序或比对旧实现，请使用：
  - `02_Core/`

## 开发原则

本项目开发遵循严谨工程规范，重点包括：

- 业务逻辑与 IO 层分离
- 配置外部化管理
- 可维护性与可追溯性优先
- 对旧系统的兼容与平滑迁移
- 开发理念受到包豪斯风格影响
