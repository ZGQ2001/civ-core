# 里氏硬度原始数据 Excel 格式规范

> 适用：钢结构里氏硬度检测原始记录。  
> 模板：[`templates/leeb_hardness/原始数据模板.xlsx`](../../../templates/leeb_hardness/原始数据模板.xlsx)  
> 锁定日期：2026-05-20

---

## 1. 文件粒度

**一个 xlsx 文件 = 一个检测项目实例**（如「里氏硬度-D号站房.xlsx」）。

- 同一文件内可混装钢柱 / 钢梁 / 不同构件类型 —— 按**检测批**区分（每批一个 sheet）。
- 不同项目之间应独立成文件。

## 2. Sheet 约定

**一个 sheet = 一个检测批**。

- Sheet 名 = 检测批名（如「检测批1」「检测批2」「钢柱-D栋」等中文名都行）。
- Sheet 顺序 = 检测批顺序。
- 不属于检测数据的 sheet（如「委托信息」）会被 UI 自动跳过（用 `sheet_name_filter='检测批'` 过滤）。

## 3. Sheet 内列布局（固定）

| 列  | 名称       | 类型 | 说明                                    |
|-----|-----------|------|----------------------------------------|
| A   | 序号      | int  | 每构件首行填，后续测区行留空            |
| B   | 构件位置  | str  | 同 A：仅首行填，后续测区行留空          |
| C-K | HL1..HL9  | int  | 9 个里氏硬度读数（每测区一行，共 3 行） |
| L   | 厚度(mm)  | num  | 同 A：仅首行填，后续测区行留空          |

**每构件占 3 行**：
- 第 1 行：序号 + 构件名 + 测区 1 的 9 点 + 厚度
- 第 2 行：测区 2 的 9 点
- 第 3 行：测区 3 的 9 点

第 1 行必须从 sheet 第 2 行起（第 1 行是表头）。构件之间不留空行。

## 4. 不在原始数据里的信息

| 信息            | 怎么传                                          |
|----------------|------------------------------------------------|
| 测量角度        | UI 顶栏全局选择（默认 0° 水平）                |
| 设计强度等级    | 当前版本不收（推定值由人工对照规范判定）       |
| 检测项目名     | 文件名自带（无需在表内重复）                    |
| 工程信息        | 可放单独的「委托信息」sheet（UI 不解析）        |

## 5. 角度档（UI 顶栏下拉）

| 度数  | 物理含义                              |
|-------|--------------------------------------|
| -90° | 向下垂直 ↓ —— **规范基线档**（HL_a 全档 = 0）|
| -45° | 向下 45°                             |
| **0°** | **水平 →（默认工作角度）**          |
| +45° | 向上 45°                             |
| +90° | 向上垂直 ↑ —— 最大修正档（HL_a 最负值）|

## 6. 结果文件（导出）

UI 「导出 Excel」生成的结果文件**独立于原始档案**，命名建议加 `-结果`。

- 每检测批生成 **2 个 sheet**：
  - `<批名>-过程数据`：每测区一行，含 HL_m / HL_t / HL_a / HL_corr / fb_min / fb_max + 构件聚合
  - `<批名>-报告插入表`：仿 D 号站房报告风格，可直接拷贝粘报告 Word

## 7. 校验规则（导入时）

- 9 个 HL 值缺一即报错；
- 厚度列必须 > 0；
- 序号列首行必填、必须为整数；
- 构件位置必填（不可为空字符串）；
- 测区行数 = 3（少于 3 也可读但 UI 会警告）。

## 8. 模板使用

1. 复制 `templates/leeb_hardness/原始数据模板.xlsx` 到项目目录
2. 重命名（如「里氏硬度-A号厂房.xlsx」）
3. 改 sheet 名（「检测批1」→ 你实际的批名）
4. 填数据（行结构必须 3 行/构件）
5. 在 GUI「里氏硬度」工具页 → 导入 → 选角度 → 计算 → 导出
