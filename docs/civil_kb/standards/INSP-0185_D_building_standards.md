## 六、注意事项

1. **版本混用**：GB/T 709 在判定依据列出2019版，但正文构件截面尺寸检测方法实际引用 GB/T 709-2006（§6.1.1 表2）。GB/T 709-2006 已被2019版替代。
2. **标准名称及编号错误**：报告正文钢筋保护层厚度抽样原则引用"《混凝土结构工程质量验收规范》（GB/T 50204-2015）"，该标准正式名称为《混凝土结构工程施工质量验收规范》，且为强制性标准 GB（非推荐性 GB/T）。
3. **新增标准**：0185报告首次引入 JGJ/T 152-2019（钢筋检测）和 DB11/T 365-2016（北京地标），扩展了钢筋检测方面的规范覆盖。
4. **无抗震标注**：0185报告首页未明确标注抗震设计依据，报告表格中无相关字段。
