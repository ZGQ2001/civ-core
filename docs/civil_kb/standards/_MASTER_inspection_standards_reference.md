# 检测项目 × 规范速查手册

> **用法**：按"检测项目"查表 → 直接获得 抽样原则 / 检测方法 / 判定标准 → 定位到规范具体条款。
> **数据补充来源**：通用表格模板（鉴定报告模板合集），补充了砌体结构检测、鉴定评级、钻芯法、贯入法等内容。

---

## ⚠️ 使用场景前置声明（必读）

本手册涉及两类完全不同的工程场景，抽样逻辑与判定标准不可混用：

| 维度 | **[A] 新建施工验收** | **[B] 既有建筑鉴定** |
|------|---------------------|---------------------|
| **管辖对象** | 在建或新建工程，按检验批逐批验收 | 已建成并投入使用的建筑 |
| **核心规范** | GB 50204（混验规）、GB 50205（钢验标）等验收类规范 | GB/T 50344（结检标）、GB 55021（鉴通规）、DB11/T 637（安鉴标）等鉴定类规范 |
| **抽样逻辑** | 按检验批总量取百分比（如 10% 且 ≥3 件），关注代表性 | 按检测批区分 A/B/C 类，最小样本容量由统计公式确定（GB/T 50344 §3.4.4），关注推断精度 |
| **判定逻辑** | **合格 / 不合格** 二值判定；尺寸超标 → 不合格，需返工 | **承载力验算 / 评级**；尺寸超标 → 不能直接判"构件不合格"，应按实际尺寸计入承载力校核，最终以 au/bu/cu/du 评级表达 |

**通用规范的凌驾性**：
- `GB 55008-2021`（混通规）和 `GB 55021-2021`（鉴通规）为**全文强制性规范**。
- 所有检测项目的抽样数量、判定底线**不得低于通规要求**。通规与推荐性标准冲突时，**以通规为准**。
- 推荐性标准（GB/T、JGJ/T）提供技术路径，通规设定法定底线——两条线需同时满足。

**本手册表格阅读约定**：
- 每项检测均区分 **[A. 新建验收]** 和 **[B. 既有鉴定]** 两种属性。无标注的属性表示两者共用。
- 若项目中仅适用一种属性，则仅列出该属性的内容。

## 一、钢结构检测项目

### 1. 轴线尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 建筑轴线间距（东西向跨度、南北向柱距） |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |
| **抽样原则** | **[A]** **钢验标 §9.2.3**：抽取 10% 且不少于 3 件；报告实际操作：划分 1 个检测批<br>**[B]** **结检标 GB/T 50344 §3.4.4**：按 B 类检测类别确定最小样本容量；且需满足 **鉴通规 GB 55021** 底线要求 |
| **检测方法** | 激光测距仪 + 钢卷尺 实测轴线间距 |
| **判定与取值** | **[A]** **钢验标 §9.2.3**：轴线尺寸允许偏差 **±4mm** → 合格/不合格<br>**[B]** 不直接做合格/不合格判定；按实际轴线尺寸计入结构分析模型，用于后续承载力验算或变形评估 |
| **避坑提醒** | 既有建筑轴线偏差 ≠ 构件不合格。鉴定报告应表述为"实测轴线尺寸为 xxx"，而非"轴线尺寸不合格"。 |

### 2. 构件截面尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁、钢柱、柱间支撑、屋面檩条的截面高度/宽度/厚度 |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |
| **抽样原则** | **[A]** **钢验标 §9.3.9**：钢梁、钢柱各抽取 10% 且不少于 3 件；报告按构件类型划分多个检测批<br>**[B]** **结检标 GB/T 50344 §3.4.4**：按构件类型划分检测批，B 类检测确定样本容量；且需满足 **鉴通规 GB 55021** 底线要求 |
| **检测方法** | 钢卷尺、游标卡尺、超声波测厚仪 |
| **判定与取值** | **[A] 验收**：按各产品标准允许偏差判定合格/不合格。<br>**焊制H型钢**：**焊H标 GB/T 33814** — H<500 → ±2.0mm; 500≤H<1000 → ±3.0mm; ≥1000 → ±4.0mm<br>**热轧H型钢**：**H型钢标 GB/T 11263** — H<400 → ±2.0mm; 400≤H<600 → ±3.0mm; 600≤H<1000 → ±4.0mm<br>**热轧工字钢/角钢**：**型钢尺 GB/T 706** 表 2<br>**冷弯Z形钢**：**冷弯尺 GB/T 6723** §5.4<br>**无缝钢管**：**无缝管标 GB/T 8162** — 外径 ±1%D 或 ±0.5mm<br>**钢材厚度**：**钢板尺 GB/T 709** §6.1.1<br><br>**[B] 鉴定**：**混现检 GB/T 50784 §8.2.3**（逻辑参照）：偏差 ≤ 允许值 1.5 倍时按设计尺寸计算承载力；偏差 > 1.5 倍时按**实测最小值**计入承载力验算。最终由承载力结果确定构件评级（au/bu/cu/du）。 |
| **关键限值速记** | H 形截面高度: h<500 → ±2.0mm; 500≤h<1000 → ±3.0mm; ≥1000 → ±4.0mm<br>板材厚度: t(3~5] → ±0.45mm; t(5~8] → ±0.50mm; t(8~15] → ±0.55mm; t(15~25] → ±0.65mm<br>等边角钢宽度: b≤56 → ±0.8mm; 56＜b≤90 → ±1.2mm; 90＜b≤140 → ±1.8mm<br>📋 完整公差表：[检测 SOP-01](inspection_sop/01_steel_section_dimensions.md) |
| **避坑提醒** | 既有鉴定严禁直接因截面尺寸超验收规范公差判"构件不合格"。鉴定关注的是实际尺寸能否通过承载力校核。 |

### 3. 钢材强度检测（里氏硬度法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁、钢柱、柱间支撑的钢材抗拉强度 |
| **检测属性** | **[B. 既有鉴定]**（新建验收以材质证明文件为准，现场硬度检测主要用于既有建筑） |
| **抽样原则** | **钢现检 GB/T 50621 §3.4.4**：按 B 类检测类别确定最小样本容量；报告按构件类型划分检测批 |
| **检测方法** | 里氏硬度计（如 TH160）测量 → **结检标 GB/T 50344 附录 N**：表面硬度法换算为抗拉强度特征值<br>原理：冲击体弹力冲击试样表面，测量 1mm 处冲击速度与回弹速度比值 |
| **判定与取值** | 推定抗拉强度特征值 ≥ 设计图纸要求的钢材强度等级（刚架梁/钢柱 → Q355；柱间支撑 → Q235）<br>**鉴定**：参见 **鉴通规 GB 55021** 及 **安鉴标 DB11/T 637** 有关条款 |
| **避坑提醒** | 里氏硬度法为**间接推定方法**，当推定值处于设计强度等级临界附近或有争议时，应辅以取样拉伸试验（GB/T 2975）验证。 |

### 4. 构件连接检测（高强螺栓）

| 事项 | 内容 |
|------|------|
| **检测对象** | 梁柱连接节点、梁梁拼接节点的高强螺栓规格和数量 |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |
| **抽样原则** | **[A]** **钢验标 GB 50205** 有关规定<br>**[B]** **检测鉴定规 DB11/T 849 §5.4.5**：螺栓间距/边距、连接板尺寸，检测数量不少于 3 件 |
| **检测方法** | 钢直尺 + 数显卡尺 + 超声波测厚仪；参照 **钢验标 GB 50205** 有关规定 |
| **判定与取值** | **[A]** 螺栓规格（如 M24）、数量（8/10/14 等）与设计图纸一致 → 合格<br>**[B]** 螺栓规格、数量与设计图纸一致 → 构造符合要求；配合节点承载力验算综合评定 |

### 5. 钢构件防腐涂层厚度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 钢构件防腐涂层干漆膜厚度 |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |
| **抽样原则** | **钢现检 GB/T 50621**：每处 3 个测点 |
| **检测方法** | 涂层测厚仪 |
| **判定与取值** | **[A]** **钢现检 GB/T 50621** — 每处 3 测点平均值 ≥ 设计厚度 85%；构件 15 测点平均值 ≥ 设计值<br>设计无要求时：室外 ≥150μm，室内 ≥125μm，允许偏差 -25μm<br>**[B]** 评估涂层剩余防护寿命与锈蚀程度；鉴定报告关注"涂层是否仍在有效防护"，而非单纯厚度合格率 |
| **避坑提醒** | 既有鉴定中，防腐涂层已局部脱落属正常老化现象，不应按验收标准判"不合格"——应描述现状并评估对构件耐久性的影响。 |

### 6. 钢构件防火涂层厚度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 钢梁（超薄型 ~2mm 设计值）、钢柱（厚型 ~20mm 设计值）的防火涂层厚度 |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |
| **抽样原则** | **钢验标 GB 50205 §13.4.3 / §13.4.4**：总数 10% 且不少于 3 个 |
| **检测方法** | **超薄型/薄型**：涂层测厚仪（**钢现检 GB/T 50621 §12**）<br>**厚型**：探针（**钢现检 GB/T 50621 §13.1~§13.4**）<br>注：超薄型也可参照 **防火涂规 DB11/1245 附录 B.0.1** |
| **判定与取值** | **[A]** **钢现检 GB/T 50621** — 同一截面各测点平均值 ≥ 设计厚度 85%；构件所有测点平均值 ≥ 设计值<br>**[B]** 评估防火涂层现状是否仍满足耐火极限要求；局部脱落或减薄需评估对整体防火性能的影响 |

### 7. 钢梁挠度变形检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁的跨中挠度 |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |

|------|------|

### 10. 混凝土轴线尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土框架/预制柱区域的轴线间距 |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |
| **抽样原则** | **[A]** **混验规 §9.3.9** — 划分 10 个检测批<br>**混验规 §8.3.2** — 划分 5 个检测批<br>**[B]** **结检标 GB/T 50344 §3.4.4**：按 B 类检测确定样本容量；且需满足 **混通规 GB 55008**、**鉴通规 GB 55021** 底线要求 |

### 11. 混凝土抗压强度检测（回弹法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 预制/现浇混凝土柱、梁的混凝土抗压强度 |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |
| **抽样原则** | **[A]** **混现检 GB/T 50784 §3.4.4**：确定检测批最小样本容量<br>**[B]** 同上；且需满足 **混通规 GB 55008** 对混凝土强度推定的底线要求 |
| **检测方法** | 回弹仪，依据 **回弹规 JGJ/T 23 第 7 章**：回弹值 → 换算强度 → 推定区间 |
| **判定与取值** | **[A]** **混验规 GB 50204**：推定值 ≥ 设计强度等级 → 合格<br>**[B]** 推定值用于承载力验算，参见 **鉴通规 GB 55021** 及 **安鉴标 DB11/T 637** |
| **⚠️ 北京地区特别提醒** | 北京地区由于骨料与掺合料特点，应用 **JGJ/T 23** 全国统一测强曲线时推定结果可能与实际偏差较大。应关注：<br>1) 是否存在北京地方专用回弹测强曲线或泵送混凝土修正系数；<br>2) 当推定值处于设计强度等级临界附近、或有争议、或需精确评定时，**必须采用钻芯法修正**（**钻芯规 JGJ/T 384**），以芯样抗压结果为最终依据。 |

### 12. 混凝土抗压强度检测（钻芯法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土构件的抗压强度（回弹法有争议或需精确评定时） |
| **检测属性** | **[A. 新建验收]** / **[B. 既有鉴定]** |
| **抽样原则** | **[A]** **钻芯规 JGJ/T 384** 有关规定<br>**[B]** 同上；且需满足 **混通规 GB 55008**、**鉴通规 GB 55021** 底线要求 |
| **检测方法** | 钻芯机取芯 → 实验室抗压试验 |
| **判定与取值** | 推定区间上限系数 k₁ 和下限系数 k₂ **按 JGJ/T 384 附录 A 查表确定，与样本容量 n 及推定方法（上限/下限）直接相关，不可固化数值**。<br>示例：n=3 时 k₁≈1.114, k₂≈2.566；n=6 时 k₁≈1.559, k₂≈2.336；n=9 时 k₁≈1.713, k₂≈2.229。n 越大，推定区间越窄（精度越高）。<br>推定值 ≥ 设计强度等级。<br>**⚠️ 报告必须注明实际 n 值及对应查表系数，严禁笼统写"k₁=1.11397, k₂=2.56600"。** |
| **避坑提醒** | k₁/k₂ 写死为 n=3 的特例值是常见报告错误。若现场钻了 6 个芯，还用 n=3 的系数，推定结果可能存在系统性偏差，报告审查时极易被退回。 |

### 13. 钢筋配置检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土构件中钢筋数量、间距 |

## 四、鉴定评级速查

> 依据《房屋结构综合安全性鉴定标准》DB11/T 637-2024（**安鉴标**）。

### 安全性鉴定评级

| 级别 | 含义 | 说明 |
|------|------|------|
| **au** | 安全 | 构件承载能力满足要求，构造与连接符合规范 |
| **bu** | 基本安全 | 构件承载能力略低于要求，但不影响整体安全性 |
| **cu** | 不安全 | 构件承载能力不满足要求，需采取措施 |
| **du** | 极不安全 | 构件严重不满足要求，存在安全隐患 |

**构件评级路径**：

- 承载能力评级（**安鉴标 §4.x**）→ **au / bu / cu / du**
- 构造与连接评级 → **au / bu / cu / du**
- 位移或变形评级 → **au / bu / cu / du**
- 裂缝或其他损伤评级 → **au / bu / cu / du**
- 取最低级别作为构件安全性等级

**子系统评级**：

- 主体结构子系统按承载功能评级 → **Au / Bu / Cu / Du**
- 围护结构子系统评级 → **Au / Bu / Cu / Du**

**鉴定系统（房屋整体）评级**：

- **Asu** — 满足安全性要求，不影响整体安全性能
- **Bsu** — 基本满足安全性要求
- **Csu** — 不满足安全性要求
- **Dsu** — 严重不满足安全性要求

---

### 抗震鉴定评级

| 级别 | 含义 | 说明 |
|------|------|------|
| **Ae** | 抗震能力满足要求 | 抗震措施和抗震承载力均满足 |
| **Be** | 基本满足抗震要求 | 略有不足但不影响整体抗震性能 |
| **Ce** | 不满足抗震要求 | 需进行抗震加固 |
| **De** | 严重不满足抗震要求 | 存在严重抗震安全隐患 |

**抗震鉴定依据**：

- A/B 类建筑 → **抗震鉴标 GB 50023** 或 **安鉴标 DB11/T 637 附录 E**
- C 类建筑 → **抗规 GB/T 50011**（2016 版在设计中使用）

**抗震措施检查要点**（砌体结构示例）：

- 房屋总高度和层数
- 横墙最大间距
- 圈梁布置与构造
- 构造柱设置
- 屋面型式（宜采用轻型屋盖）
- 外墙尽端至门窗洞边最小宽度

**鉴定系统抗震评级**：

- **Ase** — 满足现行抗震要求，不影响整体抗震性能
- **Bse** — 基本满足抗震要求
- **Cse** — 不满足抗震要求
- **Dse** — 严重不满足抗震要求

**综合安全性等级**：

- **Aeu** — 安全性和抗震能力均满足要求，在后续工作年限内不影响整体承载和抗震性能

## 六、回弹仪选用指南
