## 一、钢结构检测项目

### 1. 轴线尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 建筑轴线间距（东西向跨度、南北向柱距） |
| **抽样原则** | **钢验标 §9.2.3**：抽取 10% 且不少于 3 件；报告实际操作：划分 1 个检测批 |
| **检测方法** | 激光测距仪 + 钢卷尺 实测轴线间距 |
| **判定标准** | **验收、检测**：**钢验标 §9.2.3**：轴线尺寸允许偏差 **±4mm** |

---

### 2. 构件截面尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁、钢柱、柱间支撑、屋面檩条的截面高度/宽度/厚度 |
| **抽样原则** | **钢验标 §9.3.9**：钢梁、钢柱各抽取 10% 且不少于 3 件；报告按构件类型划分多个检测批 |
| **检测方法** | 钢卷尺、游标卡尺、超声波测厚仪 |
| **判定标准** | **检测、验收**：<br>**焊制H型钢**：**焊H标 GB/T 33814** — H<500 → ±2.0mm; 500≤H<1000 → ±3.0mm; ≥1000 → ±4.0mm<br>**热轧H型钢**：**H型钢标 GB/T 11263** — H<400 → ±2.0mm; 400≤H<600 → ±3.0mm; 600≤H<1000 → ±4.0mm<br>**热轧工字钢/角钢**：**型钢尺 GB/T 706** 表 2<br>**冷弯Z形钢**：**冷弯尺 GB/T 6723** §5.4<br>**无缝钢管**：**无缝管标 GB/T 8162** — 外径 ±1%D 或 ±0.5mm<br>**钢材厚度**：**钢板尺 GB/T 709** §6.1.1 |
| **关键限值速记** | H 形截面高度: h<500 → ±2.0mm; 500≤h<1000 → ±3.0mm; ≥1000 → ±4.0mm<br>板材厚度: t(3~5] → ±0.45mm; t(5~8] → ±0.50mm; t(8~15] → ±0.55mm; t(15~25] → ±0.65mm<br>等边角钢宽度: b≤56 → ±0.8mm; 56＜b≤90 → ±1.2mm; 90＜b≤140 → ±1.8mm<br>📋 完整公差表：[检测 SOP-01](inspection_sop/01_steel_section_dimensions.md) |

---

### 3. 钢材强度检测（里氏硬度法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁、钢柱、柱间支撑的钢材抗拉强度 |
| **抽样原则** | **钢现检 GB/T 50621 §3.4.4**：按 B 类检测类别确定最小样本容量；报告按构件类型划分检测批 |
| **检测方法** | 里氏硬度计（如 TH160）测量 → **结检标 GB/T 50344 附录 N**：表面硬度法换算为抗拉强度特征值<br>原理：冲击体弹力冲击试样表面，测量 1mm 处冲击速度与回弹速度比值 |
| **判定标准** | **检测**：设计图纸要求的钢材强度等级：刚架梁/钢柱 → Q355；柱间支撑 → Q235<br>**鉴定**：参见 GB 50292 / GB 55021 有关条款 |

---

### 4. 构件连接检测（高强螺栓）

| 事项 | 内容 |
|------|------|
| **检测对象** | 梁柱连接节点、梁梁拼接节点的高强螺栓规格和数量 |
| **抽样原则** | **检测鉴定规 DB11/T 849 §5.4.5**：螺栓间距/边距、连接板尺寸，检测数量不少于 3 件 |
| **检测方法** | 钢直尺 + 数显卡尺 + 超声波测厚仪；参照 **钢验标 GB 50205** 有关规定 |
| **判定标准** | **检测、验收**：设计图纸：螺栓规格（如 M24）、数量（8/10/14 等）与图纸一致即为符合要求 |

---

### 5. 钢构件防腐涂层厚度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 钢构件防腐涂层干漆膜厚度 |
| **抽样原则** | **钢现检 GB/T 50621**：每处 3 个测点 |
| **检测方法** | 涂层测厚仪 |
| **判定标准** | **检测、验收**：**钢现检 GB/T 50621** — 每处 3 测点平均值 ≥ 设计厚度 85%；构件 15 测点平均值 ≥ 设计值<br>设计无要求时：室外 ≥150μm，室内 ≥125μm，允许偏差 -25μm |

---

### 6. 钢构件防火涂层厚度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 钢梁（超薄型 ~2mm 设计值）、钢柱（厚型 ~20mm 设计值）的防火涂层厚度 |
| **抽样原则** | **钢验标 GB 50205 §13.4.3 / §13.4.4**：总数 10% 且不少于 3 个 |
| **检测方法** | **超薄型/薄型**：涂层测厚仪（**钢现检 GB/T 50621 §12**）<br>**厚型**：探针（**钢现检 GB/T 50621 §13.1~§13.4**）<br>注：超薄型也可参照 **防火涂规 DB11/1245 附录 B.0.1** |
| **判定标准** | **检测、验收**：**钢现检 GB/T 50621** — 同一截面各测点平均值 ≥ 设计厚度 85%；构件所有测点平均值 ≥ 设计值 |

---

### 7. 钢梁挠度变形检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁的跨中挠度 |
| **抽样原则** | **钢验标 GB 50205 §10.4.1**：总数 10% 且不少于 3 个 |
| **检测方法** | 全站仪（如 TZ08）测量"端部-跨中-端部"3 个测点标高，计算挠度值 |
| **判定标准** | **检测、验收**：**钢设标 GB 50017 表 B.1.1**：挠度容许值 **< l₀/400**（例：20m 跨 → 50mm；12m 跨 → 30mm）<br>**鉴定**：参见 GB 50292 / GB 55021 有关条款 |

---

### 8. 钢柱轴线垂直度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 钢柱在东西向、南北向的倾斜 |
| **抽样原则** | **钢验标 GB 50205 §10.3.4**：按 10% 抽查且不少于 3 件 |
| **检测方法** | 全站仪，依据 **变测规 JGJ 8** 中有关规定 |
| **判定标准** | **检测、验收**：**钢验标 GB 50205 表 10.3.4**：倾斜率 **≤ 1/1000**，且不应大于 **10.0mm** |

---

### 9. 外观质量检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 建筑外观装饰装修面、结构构件表面、焊缝外观 |
| **抽样原则** | **结检标 GB/T 50344 §3.3.9 / §4.5**：全数检测 |
| **检测方法** | 目测 + 尺量：检查构件弯曲变形、裂纹、锈蚀；焊缝裂纹、未焊满、咬边、表面气孔、夹渣 |
| **判定标准** | **检测**：无主要承重构件弯曲变形/裂纹/锈蚀；无焊缝外观损伤 → 合格<br>**鉴定**：参见 GB 50292 / GB 55021 有关条款 |

## 二、混凝土结构检测项目

### 10. 混凝土轴线尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土框架/预制柱区域的轴线间距 |
| **抽样原则** | **混验规 §9.3.9** — 划分 10 个检测批<br>**混验规 §8.3.2** + **钢验标 §9.2.3** 双引用 — 划分 5 个检测批 |
| **检测方法** | 激光测距仪 + 钢卷尺 实测轴线间距（与钢结构轴线检测设备相同） |
| **判定标准** | **检测、验收**：**混验规 表 9.3.9** → 允许偏差 **±8mm**<br>**混验规 表 8.3.2** → 允许偏差 **±8mm**<br>（钢结构轴线部分仍按钢验标 → ±4mm） |

### 11. 混凝土抗压强度检测（回弹法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 预制/现浇混凝土柱、梁的混凝土抗压强度 |
| **抽样原则** | **混现检 GB/T 50784 §3.4.4**：确定检测批最小样本容量 |
| **检测方法** | 回弹仪，依据 **回弹规 JGJ/T 23 第 7 章**：回弹值 → 换算强度 → 推定区间 |
| **判定标准** | **检测、验收**：**混验规 GB 50204**：推定值 ≥ 设计强度等级<br>**鉴定**：参见 GB 50292 / GB 55021 有关条款 |

---

### 12. 混凝土抗压强度检测（钻芯法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土构件的抗压强度（回弹法有争议或需精确评定时） |
| **抽样原则** | **钻芯规 JGJ/T 384** 有关规定 |
| **检测方法** | 钻芯机取芯 → 实验室抗压试验 |
| **判定标准** | **检测、验收**：**钻芯规 JGJ/T 384 §6.3.2 / 附录 A** — 推定区间上限系数 k₁=1.11397，下限系数 k₂=2.56600<br>推定区间的上限值和下限值按公式计算；推定值 ≥ 设计强度等级 |

### 13. 钢筋配置检测

### 15. 混凝土构件截面尺寸检测

## 三、砌体结构检测项目

> 砌体结构检测方法主要依据《砌体工程现场检测技术标准》GB/T 50315-2011（**砌检标**）。

### 17. 烧结普通砖抗压强度检测（回弹法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 承重墙体烧结普通砖抗压强度 |
| **抽样原则** | **砌检标 GB/T 50315** 有关规定（按检测批划分） |
| **检测方法** | 回弹仪，依据 **砌检标 GB/T 50315 §14.4**：回弹值 → 换算强度 → 评定 |
| **判定标准** | **检测**：推定结果 ≥ 设计强度等级（如 MU10）<br>**鉴定**：参见 **安鉴标 DB11/T 637** 有关条款 |

---

### 18. 砌筑砂浆抗压强度检测（回弹法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 承重墙体砌筑砂浆抗压强度 |
| **抽样原则** | **砌检标 GB/T 50315** 有关规定 |
| **检测方法** | 回弹仪，依据 **砌检标 GB/T 50315 §12.4 / §15.0.4~§15.0.5** |
| **判定标准** | **检测**：推定结果 ≥ 设计强度等级（如 M5）<br>**鉴定**：参见 **安鉴标 DB11/T 637** 有关条款 |

---

### 19. 砌筑砂浆抗压强度检测（贯入法）

| 事项 | 内容 |
|------|------|

| **抽样原则** | **贯入规 JGJ/T 136** 有关规定 |
| **检测方法** | 贯入仪，依据 **贯入规 JGJ/T 136 §5.0.5**：贯入深度 → 换算强度 → 评定 |
| **判定标准** | **检测**：推定结果 ≥ 设计强度等级（如 M5）<br>**鉴定**：参见 **安鉴标 DB11/T 637** 有关条款 |

---

### 20. 墙体高厚比检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 承重墙体的计算高度与厚度比值 |
| **抽样原则** | 全数或按检测批 |
| **检测方法** | 钢卷尺/激光测距仪实测墙高与墙厚，计算高厚比 |
| **判定标准** | **检测、鉴定**：**GB 50003 §6.1.1** — 实际高厚比 ≤ 允许高厚比<br>高厚比满足则评为 au 级构件（**安鉴标 DB11/T 637**） |

### 21. 建筑整体倾斜检测

## 四、鉴定评级速查

> 依据《房屋结构综合安全性鉴定标准》DB11/T 637-2024（**安鉴标**）。

### 安全性鉴定评级

| 级别 | 含义 | 说明 |
|------|------|------|
| **au** | 安全 | 构件承载能力满足要求，构造与连接符合规范 |
| **bu** | 基本安全 | 构件承载能力略低于要求，但不影响整体安全性 |
| **cu** | 不安全 | 构件承载能力不满足要求，需采取措施 |
| **du** | 极不安全 | 构件严重不满足要求，存在安全隐患 |

**构件评级路径**：
- 承载能力评级（**安鉴标 §4.x**）→ **au / bu / cu / du**
- 构造与连接评级 → **au / bu / cu / du**
- 位移或变形评级 → **au / bu / cu / du**
- 裂缝或其他损伤评级 → **au / bu / cu / du**
- 取最低级别作为构件安全性等级

**子系统评级**：
- 主体结构子系统按承载功能评级 → **Au / Bu / Cu / Du**
- 围护结构子系统评级 → **Au / Bu / Cu / Du**

**鉴定系统（房屋整体）评级**：
- **Asu** — 满足安全性要求，不影响整体安全性能
- **Bsu** — 基本满足安全性要求
- **Csu** — 不满足安全性要求
- **Dsu** — 严重不满足安全性要求

---

### 抗震鉴定评级

| 级别 | 含义 | 说明 |
|------|------|------|
| **Ae** | 抗震能力满足要求 | 抗震措施和抗震承载力均满足 |
| **Be** | 基本满足抗震要求 | 略有不足但不影响整体抗震性能 |
| **Ce** | 不满足抗震要求 | 需进行抗震加固 |
| **De** | 严重不满足抗震要求 | 存在严重抗震安全隐患 |

**抗震鉴定依据**：
- A/B 类建筑 → **抗震鉴标 GB 50023** 或 **安鉴标 DB11/T 637 附录 E**
- C 类建筑 → **抗规 GB/T 50011**（2016 版在设计中使用）

**抗震措施检查要点**（砌体结构示例）：
- 房屋总高度和层数
- 横墙最大间距
- 圈梁布置与构造
- 构造柱设置
- 屋面型式（宜采用轻型屋盖）
- 外墙尽端至门窗洞边最小宽度

**鉴定系统抗震评级**：
- **Ase** — 满足现行抗震要求，不影响整体抗震性能
- **Bse** — 基本满足抗震要求
- **Cse** — 不满足抗震要求
- **Dse** — 严重不满足抗震要求

**综合安全性等级**：
- **Aeu** — 安全性和抗震能力均满足要求，在后续工作年限内不影响整体承载和抗震性能

## 七、钢筋保护层厚度检测（详细）

> 本项补充展开 #13、#14 的检测方法细节。

| 事项 | 内容 |
|------|------|
| **检测对象** | 梁底/板底/悬挑构件受力钢筋保护层厚度 |
| **抽样原则** | **GB 50204-2015 附录 E**：非悬挑梁板各 **2% 且 ≥5 个**；悬挑梁 **5% 且 ≥10 个**；悬挑板 **10% 且 ≥20 个**。梁底面纵向受力钢筋全部检测；板类 ≥6 根纵向受力钢筋，每根 3 测点取平均 |
| **检测方法** | 电磁感应法钢筋扫描仪，依据 **JGJ/T 152-2019** / **DB11/T 365-2016** |
| **关键操作** | 1) 探头长边平行于待测钢筋轴线缓慢移动；2) 每根钢筋连续测两次（正反向各一次），两次差 ≤±1mm 有效；3) 相邻钢筋净间距 > 保护层厚度时才能有效识别；4) 第一量程 5~60mm，第二量程 60~180mm |
| **允许偏差** | **梁类**：**+10, -7** mm；**板类**：**+8, -5** mm。超限但 ≤1.5 倍允许偏差（梁+15/-10.5，板+12/-7.5）→ 仍可判合格 |
| **判定标准** | **GB 50204-2015 附录 E**：合格率 ≥ 90% → 合格；80%~90% → 加倍抽样合并计算；< 80% → 不合格。GB/T 50344-2019 按表 3.5.3-1/3.5.3-2 计数抽样判定 |
| **结论模板** | "本次检测的 XX 工程 X 片梁钢筋保护层厚度共计 X 根钢筋 X 个测点，合格 X 根钢筋 X 个测点，合格率为 XX%。不合格点最大偏差不大于允许偏差的 1.5 倍，检测结果为合格。" |

## 八、钢筋锈蚀性状检测（半电池电位法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土中钢筋锈蚀性状 |
| **抽样原则** | **GB/T 50344-2019 附录 L.0.5**：按环境差异及外观分类，每类 ≥3 个测区；测区 ≤5m×5m，网格 100~500mm，≥20 个测点 |
| **检测方法** | 半电池电位法（铜-硫酸铜参比电极），依据 **JGJ/T 152-2019** |
| **检测原理** | 钢筋钝化状态 → 电位偏正；活化（锈蚀）状态 → 电位偏负；最大电位差可达 500mV |
| **操作要点** | 1) 配制过饱和硫酸铜溶液；2) 混凝土表面浸湿处理；3) 局部剔凿钢筋连接电压仪负端；4) 记录环境温度进行修正（22±5℃）；5) 绘制电位等值线图（间隔 ≤100mV） |
| **判定标准** | **JGJ/T 152-2019**：电位 > -200 mV → 不发生锈蚀概率 > 90%；-200~-350 mV → 锈蚀性状不确定；< -350 mV → 发生锈蚀概率 > 90%。辅以局部剔凿验证 |
| **设备维护** | 铜棒用稀盐酸轻擦清洗（禁用钢毛刷）；硫酸铜溶液约半年更换一次，更换后甘汞电极校准 |
