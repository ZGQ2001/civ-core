## 一、钢结构检测项目

### 1. 轴线尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 建筑轴线间距（东西向跨度、南北向柱距） |
| **抽样原则** | **钢验标 §9.2.3**：抽取 10% 且不少于 3 件；报告实际操作：划分 1 个检测批 |
| **检测方法** | 激光测距仪 + 钢卷尺 实测轴线间距 |
| **判定标准** | **钢验标 §9.2.3**：轴线尺寸允许偏差 **±4mm** |

---

### 2. 构件截面尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁、钢柱、柱间支撑、屋面檩条的截面高度/宽度/厚度 |
| **抽样原则** | **钢验标 §9.3.9**：钢梁、钢柱各抽取 10% 且不少于 3 件；报告按构件类型划分多个检测批 |
| **检测方法** | 钢卷尺、游标卡尺、超声波测厚仪 |
| **判定标准** | **焊制H型钢/箱型截面**：**钢验标** 表 8.3.2（高度、宽度偏差）<br>**热轧等边角钢**（如 L100×8）：**型钢尺 GB/T 706** 表 2 <br>**冷弯Z形钢**（如 XZ250×75×20×2.0）：**冷弯尺 GB/T 6723** §5.4 表 11、表 12<br>**钢材厚度**：**钢板尺 GB/T 709** §6.1.1 表 2、表 3（注意：报告实际用 2006 旧版，建议使用时升级至 2019 版） |
| **关键限值速记** | H 形截面高度: h<500 → ±2.0mm; 500≤h<1000 → ±3.0mm; ≥1000 → ±4.0mm<br>板材厚度: t(3~5] → ±0.45mm; t(5~8] → ±0.50mm; t(8~15] → ±0.55mm; t(15~25] → ±0.65mm |

---

### 3. 钢材强度检测（里氏硬度法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁、钢柱、柱间支撑的钢材抗拉强度 |
| **抽样原则** | **钢现检 GB/T 50621 §3.4.4**：按 B 类检测类别确定最小样本容量；报告按构件类型划分检测批 |
| **检测方法** | 里氏硬度计（如 TH160）测量 → **结检标 GB/T 50344 附录 N**：表面硬度法换算为抗拉强度特征值 |
| **判定标准** | 设计图纸要求的钢材强度等级：刚架梁/钢柱 → Q355；柱间支撑 → Q235 |

---

### 4. 构件连接检测（高强螺栓）

| 事项 | 内容 |
|------|------|
| **检测对象** | 梁柱连接节点、梁梁拼接节点的高强螺栓规格和数量 |
| **抽样原则** | **钢现检 GB/T 50621 §3.4.4**：按 A 类检测类别确定最小样本容量 |
| **检测方法** | 目测 + 工具实测，参照 **钢验标 GB 50205** 中有关规定 |
| **判定标准** | 设计图纸：螺栓规格（如 M24）、数量（8/10/14 等）与图纸一致即为符合要求 |

---

### 5. 钢构件防火涂层厚度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 钢梁（超薄型 ~2mm 设计值）、钢柱（厚型 ~20mm 设计值）的防火涂层厚度 |
| **抽样原则** | **钢验标 GB 50205 §13.4.3 / §13.4.4**：总数 10% 且不少于 3 个 |
| **检测方法** | **超薄型**：**防火涂规 DB11/1245 附录 B.0.1**<br>**薄型**：**钢现检 GB/T 50621 §12**<br>**厚型**：**钢现检 GB/T 50621 §13.1~§13.4**<br>仪器：涂层测厚仪（超薄型、薄型）、卡尺探针（厚型） |
| **判定标准** | 实测平均值 ≥ 设计值（超薄型 3.0mm / 薄型 3.0~7.0 mm / 厚型 > 7.0mm） |

---

### 6. 钢梁挠度变形检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 刚架梁的跨中挠度 |
| **抽样原则** | **钢验标 GB 50205 §10.4.1**：总数 10% 且不少于 3 个 |
| **检测方法** | 全站仪（如 TZ08）测量"端部-跨中-端部"3 个测点标高，计算挠度值 |
| **判定标准** | **钢设标 GB 50017 表 B.1.1**：挠度容许值 **< l₀/400**（例：20m 跨 → 50mm；12m 跨 → 30mm） |
| **适用报告** | 0181 / 0183 / 0184 / 0185 |

---

### 7. 钢柱轴线垂直度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 钢柱在东西向、南北向的倾斜 |
| **抽样原则** | **钢验标 GB 50205 §10.3.4**：按 10% 抽查且不少于 3 件 |
| **检测方法** | 全站仪，依据 **变测规 JGJ 8** 中有关规定 |
| **判定标准** | **钢验标 GB 50205 表 10.3.4**：倾斜率 **≤ 1/1000**，且不应大于 **10.0mm** |
| **适用报告** | 0181 / 0183 / 0184 / 0185 |

---

### 8. 外观质量检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 建筑外观装饰装修面、结构构件表面、焊缝外观 |
| **抽样原则** | **结检标 GB/T 50344 §3.3.9 / §4.5**：全数检测 |
| **检测方法** | 目测 + 尺量：检查构件弯曲变形、裂纹、锈蚀；焊缝裂纹、未焊满、咬边、表面气孔、夹渣 |
| **判定标准** | 无主要承重构件弯曲变形/裂纹/锈蚀；无焊缝外观损伤 → 合格 |
| **适用报告** | 0181 / 0183 / 0184 / 0185 |

## 二、混凝土结构检测项目

> 以下 6 项仅适用于含混凝土结构的 0184（B号厂房）和 0185（D号厂房）。

### 9. 混凝土轴线尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土框架/预制柱区域的轴线间距 |
| **抽样原则** | **0184**：**混验规 §9.3.9** — 划分 10 个检测批<br>**0185**：**混验规 §8.3.2** + **钢验标 §9.2.3** 双引用 — 划分 5 个检测批 |
| **检测方法** | 激光测距仪 + 钢卷尺 实测轴线间距（与钢结构轴线检测设备相同） |
| **判定标准** | **0184**：**混验规 表 9.3.9** → 允许偏差 **±8mm**<br>**0185**：**混验规 表 8.3.2** → 允许偏差 **±8mm**<br>（钢结构轴线部分仍按钢验标 → ±4mm） |
| **适用报告** | 0184 / 0185 |

---

### 10. 混凝土抗压强度检测（回弹法）

| 事项 | 内容 |
|------|------|
| **检测对象** | 预制/现浇混凝土柱、梁的混凝土抗压强度 |
| **抽样原则** | **混现检 GB/T 50784 §3.4.4**：确定检测批最小样本容量 |
| **检测方法** | 回弹仪，依据 **回弹规 JGJ/T 23 第 7 章**：回弹值 → 换算强度 → 推定区间 |
| **判定标准** | **混验规 GB 50204**：推定值 ≥ 设计强度等级 |
| **适用报告** | 0184 / 0185 |

---

### 11. 钢筋配置检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土构件中钢筋数量、间距 |
| **抽样原则** | **混验规 GB 50204 §5.5.3**：钢筋配置抽样规定 |
| **检测方法** | 电磁感应法钢筋扫描仪，依据 **钢筋检 JGJ/T 152** |
| **判定标准** | **混验规 GB 50204**：钢筋数量、间距符合设计图纸 |
| **适用报告** | 0185（0184 未单独检测此项目） |

---

### 12. 钢筋保护层厚度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 混凝土构件钢筋保护层厚度 |
| **抽样原则** | **混验规 GB 50204** 附录 E 相关规定 |
| **检测方法** | 电磁感应法，依据 **保护层规 DB11/T 365**（北京地标） |
| **判定标准** | **混验规 GB 50204 附录 E.0.4** |
| **适用报告** | 0185 |

---

### 13. 混凝土构件截面尺寸检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 预制混凝土柱截面尺寸 |
| **抽样原则** | **0184**：**混验规** — 预制混凝土柱抽取 **5%** 且不少于 3 件<br>**0185**：**混验规 附录 F** — 梁、柱抽取构件数量的 **1%**，且不应少于 3 个构件 |
| **检测方法** | 钢卷尺、游标卡尺实测 |
| **判定标准** | **0184**：**混验规 附录 F / §8.3.2**<br>**0185**：**混验规 附录 F / §8.3.2** |
| **适用报告** | 0184 / 0185 |

---

### 14. 混凝土柱垂直度检测

| 事项 | 内容 |
|------|------|
| **检测对象** | 现浇/预制混凝土柱轴线垂直度 |
| **抽样原则** | **0184**：**混验规 §9.3.9** — 预制混凝土柱按 10% 抽查且不少于 3 件<br>**0185**：**混验规 §8.3.2** — 现浇混凝土柱按 10% 抽查且不少于 3 件 |
| **检测方法** | 全站仪，依据 **变测规 JGJ 8** |
| **判定标准** | **0184**：**混验规 表 9.3.9**（预制柱）<br>**0185**：**混验规 表 8.3.2**（现浇柱） |
| **适用报告** | 0184 / 0185 |
