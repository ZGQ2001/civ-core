# 检测 SOP-01：钢构件截面尺寸检测

> **适用范围**：钢结构工程中焊接H型钢、热轧H型钢、工字钢、等边角钢的截面尺寸检测。
> **检测方法**：钢卷尺 + 游标卡尺 + 超声波测厚仪。
> **判定依据**：按构件型材类别，分别查阅对应产品标准中的尺寸允许偏差表。

---

## 依据标准

| 型材类别 | 标准编号 | 标准名称 |
|---------|---------|---------|
| 焊接H型钢 | GB/T 33814-2017 | 焊接H型钢 |
| 热轧H型钢 | GB/T 11263-2024 | 热轧H型钢和剖分T型钢 |
| 工字钢 / 等边角钢 | GB/T 706-2016 | 热轧型钢 |

---

## 检测步骤

1. **确认型材类别**：根据设计图纸确认构件的型材类型（焊接H型钢 / 热轧H型钢 / 工字钢 / 角钢）
2. **选择测点**：每构件在截面高度、宽度、腹板厚度、翼缘厚度各维度至少测量 3 个测点
3. **量测**：截面高度/宽度用钢卷尺；腹板/翼缘厚度用超声波测厚仪（必要时游标卡尺复核）
4. **记录**：记录各测点实测值，计算与公称尺寸的偏差
5. **判定**：对照下方公差表，各测点偏差 ≤ 允许偏差 → 该测点合格
6. **批判定**：按 **结检标 GB/T 50344** 对检验批做计数抽样满足性判定
