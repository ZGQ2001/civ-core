# 检测 SOP-03：涂层厚度检测（防火涂层 + 防腐涂层）

> **适用范围**：钢结构工程中防火涂层、防腐涂层的干漆膜厚度现场检测。
> **检测设备**：涂层测厚仪（薄型）、卡尺探针（厚型防火涂层）。
> **判定依据**：GB/T 50621-2010《钢结构现场检测技术标准》、GB/T 50344-2019《建筑结构检测技术标准》。

# 第一部分：防腐涂层厚度检测

## 检测步骤

### 1. 测区布置

- 每构件检测 **5 处**
- 每处检测 **3 个测点**，测点间距 **50mm**（GB/T 50344-2019 §12.1.3）
- 测点部位涂层应与钢材附着良好

### 2. 测量方法

- 使用**涂层测厚仪**（电磁感应原理或涡流原理）
- 每测点测量 3 次，取平均值作为该测点值
- 测量前需在无涂层基材上校准零点

### 3. 记录与判定

| 级别 | 判定标准 | 依据 |
|------|---------|------|
| **测点级** | 每测点测 3 次取平均 | GB/T 50621-2010 |
| **每处** | 3 测点平均值 ≥ 设计厚度 **85%** | GB/T 50621-2010 |
| **构件级** | 15 测点（5处×3点）平均值 ≥ 设计值 | GB/T 50621-2010 |
| **无设计要求** | 室外 ≥ 150μm，室内 ≥ 125μm，允许偏差 -25μm | GB/T 50621-2010 |

# 第二部分：防火涂层厚度检测

## 分类

| 类型 | 设计厚度（示例） | 检测方法 | 设备 |
|------|----------------|---------|------|
| **超薄型** | ~2mm | 涂层测厚仪 | 电磁感应式测厚仪 |
| **薄型** | 3~7mm | 涂层测厚仪 | 电磁感应式测厚仪 |
| **厚型** | ~20mm | 探针法 | 卡尺探针 |

## 检测步骤

### 1. 测区布置 — 梁、柱构件

- 在构件长度内**每隔 3m 取一个截面**
- 每个构件**不少于 2 个截面**（GB/T 50344-2019 §13.1.4）
- 每个截面的测点布置如图：

```
       柱截面测点示意              梁截面测点示意
       ┌─────┬─────┐             ┌───────────────┐
       │     │     │             │               │
       │  1  2  3  │             │  1   2   3    │
       │     │     │             │               │
       └─────┴─────┘             └───────────────┘
```

### 2. 测区布置 — 楼板、墙体

- 相邻纵、横轴线相交的面积为一个构件
- 在对角线上按**每米长度选 1 个测点**
- 每个构件**不少于 5 个测点**（GB/T 50621-2010 §13.1.3）

### 3. 测量方法

| 涂层类型 | 方法 | 操作要点 |
|---------|------|---------|
| 超薄型/薄型 | 涂层测厚仪 | 直接读数，每测点 3 次取平均 |
| 厚型 | 探针法 | 垂直插入至钢材表面，读取探入深度 |

### 4. 判定标准

| 级别 | 判定标准 | 依据 |
|------|---------|------|
| **截面级** | 同一截面各测点平均值 ≥ 设计厚度 **85%** | GB/T 50621-2010 |
| **构件级** | 构件所有测点平均值 ≥ 设计值 | GB/T 50621-2010 |

---

## 测区布置汇总对比

| 涂层类型 | 构件类型 | 测区/测点布置 | 依据 |
|---------|---------|-------------|------|
| **防腐** | 所有钢构件 | 5 处/构件，3 点/处（间距 50mm） | GB/T 50344-2019 §12.1.3 |
| **防火** | 梁、柱 | 每隔 3m 一截面，≥2 截面/构件 | GB/T 50344-2019 §13.1.4 |
| **防火** | 楼板、墙体 | 对角线上每米 1 点，≥5 点/构件 | GB/T 50621-2010 §13.1.3 |

---

## 注意事项

1. **基材校准**：涂层测厚仪必须在无涂层的同类基材上校准零点，否则读数整体偏移。
2. **粗糙度影响**：钢材表面粗糙度过大（如喷砂处理后）会导致测厚仪读数偏大，需在报告中注明。
3. **防火涂层探针**：厚型防火涂层用探针法时，需确保探针垂直于表面并到达钢材基面。
4. **防腐涂层测点间距**：3 个测点间距 50mm，不宜过远（涂层厚度可能有梯度变化）。
5. **验收判定双重要求**：每处平均 ≥ 85% 设计值 且 构件平均 ≥ 设计值，两个条件必须同时满足。
