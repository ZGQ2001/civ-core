# 检测 SOP-04：回弹法检测混凝土抗压强度

> **适用范围**：普通混凝土（含泵送混凝土）抗压强度的回弹法现场检测。
> **检测设备**：回弹仪（普通 2.207J / 高强 4.5J~5.5J）、碳化深度测量仪、钢砧。
> **判定依据**：JGJ/T 23-2011、DB11/T 1446-2017（北京泵送）、T/CECS 02:2020（团标）。

## 检测步骤

### 1. 检测前的准备

1. **收集工程资料**：工程名称、设计/委托/施工单位；构件名称、数量、混凝土类型及强度等级；水泥安定性、外加剂/掺合料品种、配合比；施工模板、浇筑/养护情况及日期；必要的设计图纸和施工记录
2. **填写委托单**：选择方法标准 → 编制检测方案 → 准备原始记录表/设备使用记录/安全交底记录
3. **设备检查**：
   - 回弹仪：绿色标签、检定日期、外观、"0点"检查
   - **检测前率定 + 检测后率定**：钢砧洛氏硬度 HRC 60±2，率定值 80±2（普通）/ 88±2（高强4.5J）/ 83±1（高强5.5J）
   - 率定时转动 4 个角度（每次 90°），各弹 3 次取平均
   - 钢卷尺/直尺精度 1mm；碳化深度仪精度 0.25mm

### 2. 回弹仪选用

**不能只看设计强度等级！** 现场先试弹估算：

| 试弹结果 | 选用回弹仪 | 适用强度 |
|---------|-----------|---------|
| 普通回弹仪回弹值 > 50，换算 > 60 MPa | 高强 4.5J 或 5.5J | 60~80 MPa |
| 普通回弹仪回弹值 < 45，换算 < 60 MPa | 普通 2.207J | 20~60 MPa |
| 普通回弹仪回弹值 45~50，换算 60 左右 | 4.50J 高强 | 20~110 MPa |

### 3. 测区布置

| 参数 | 要求 |
|------|------|
| 测区大小 | 20cm×20cm，不大于 0.04 m² |
| 测区间距 | ≤ 2 m（建议标准化：0.2/0.5/1.0/2.0m，便于溯源） |
| 距构件边缘 | 0.2~0.5 m |
| 弹击面 | 优先选**浇筑侧面**（梁必须侧面；板优先底面；墙选侧面） |
| 测点分布 | 16 个测点均匀分布，同一测点不重复弹击，避开蜂窝麻面和预埋件 |
| 首末测区 | 首末测区距边缘距离必须符合要求，不能集中在局部区域 |

### 4. 抽样要求

| 检测模式 | 抽样规则 | 依据 |
|---------|---------|------|
| 单个构件 | 按规范或合同约定 | — |
| 批量检测 | ≥ 同批总数 **30%**，且 **≥ 10 个**构件；> 30 个时可调整 | JGJ/T 23-2011 §4.1.3 |
| 最小样本容量 | 按 GB/T 50784-2013 表 3.4.4（**不能用** GB/T 50344-2019 表 3.3.10，回弹法属计量检测） | GB/T 50784-2013 |

### 5. 回弹操作

- 匀速弹击，弹击杆垂直于测试面
- 同一点不得重复弹击
- 测点均匀分布在 20×20cm 范围内
- 视频录制要求：回弹法检测全过程需**视频监控** + 数据采集过程**录屏/摄像**（京建函【2023】33号）

### 6. 碳化深度测量

| 步骤 | 要求 |
|------|------|
| 测量比例 | 不少于测区数的 **30%**；碳化深度极差 > 2mm 时**全部测区**测量 |
| 孔洞制备 | 钢钎在测区表面形成 Φ≈15mm、深度≈10mm（或大于碳化深度）的孔洞 |
| 清洁 | 皮老虎吹干净碎屑 |
| 试剂 | 1%~2% 酚酞酒精溶液（标签注明配制日期和浓度） |
| 测量位置 | 孔洞内壁上边缘滴试剂（上、左、右各测一次，不宜在下方测试） |
| 读数 | 每次精确至 0.25mm，取 3 次平均值 |
| 修约 | 最终精确至 0.5mm |

**碳化深度修约简化方法**：以 0.25mm 为中间点
- ≤0.25 → 向下靠近（如 1.17→1.0）
- \>0.25 → 向上靠近（如 1.33→1.5）
- =0.25 → 向下靠近（1.25→1.0）
- =0.75 → 向上靠近（1.75→2.0）

### 7. 强度计算

详见 `formulas/INSP-003`。关键参数：
- 测区平均回弹值：16 个值剔除 3 高 3 低，取 10 个平均
- 修正顺序：**角度 → 浇筑面 → 碳化 → 芯样**（不可颠倒）
- 碳化深度 < 0.5mm 按 0mm 处理；≥ 6.0mm 按 6.0mm 处理
- 强度推定：n < 10 → 取最小值；n ≥ 10 → 平均值 − 1.645×标准差

### 8. 数字修约规则

采用**四舍六入五看前，奇进偶不进**：

| 示例 | 结果 | 规则 |
|------|------|------|
| 35.44000 → 35.4 | 四舍 | < 5 舍去 |
| 35.48000 → 35.5 | 六入 | ≥ 6 进位 |
| 35.25001 → 35.3 | 进位 | 5 后有数则进位 |
| 35.25000 → 35.2 | 不进 | 5 前偶数不进 |
| 35.35000 → 35.4 | 进位 | 5 前奇数进位 |

## 注意事项

1. **泵送混凝土侧面检测**：梁必须侧面；板类可底面回弹后用芯样或同条件试块修正，否则结果无效
2. **率定必须双次**：检测前一次 + 检测后一次，确保设备全程处于标准状态
3. **数显回弹仪**：示值误差 ≤ 1，数据传输正常
4. **设备标签**：绿色标签表示在检定有效期内；检定日期、外观、活动能力、"0"点检查均需确认
5. **记录可追溯**：原始记录中预置测区图，标注测区间距
