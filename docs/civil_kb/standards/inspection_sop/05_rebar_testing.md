# 检测 SOP-05：混凝土中钢筋检测

> **适用范围**：混凝土构件中钢筋位置、间距、保护层厚度、钢筋直径的电磁感应法现场检测。含钢筋锈蚀半电池电位法检测。
> **检测设备**：钢筋扫描仪（电磁感应法）、半电池电位仪（铜-硫酸铜电极）。
> **判定依据**：JGJ/T 152-2019、DB11/T 365-2016、GB 50204-2015、GB/T 50344-2019。

# 第一部分：钢筋位置、间距、保护层厚度

## 检测步骤

### 1. 检测原理

电磁感应法：主机发射交变电磁场 → 钢筋产生感应电流 → 激发二次磁场 → 探头接收信号 → 定位钢筋位置并计算保护层厚度。

### 2. 检测前准备

| 项目 | 要求 |
|------|------|
| 设备校验 | 标准试块上校准，误差 ≤ ±1mm |
| 预热调零 | 提前开机，探头举在空中按确定键调零 |
| 参数预设 | 预设钢筋直径、量程范围（第一量程 5~60mm；第二量程 60~180mm）、钢筋间距 |
| 饰面层 | 清除测试部位的饰面层 |
| 垫块 | 保护层厚度较小时使用垫块 |

### 3. 抽样要求

| 构件类型 | 抽样比例 | 最少数量 | 依据 |
|---------|---------|---------|------|
| 非悬挑梁、板 | 各 2% | ≥ 5 个 | GB 50204-2015 附录 E |
| 悬挑梁 | 5% | ≥ 10 个 | GB 50204-2015 附录 E |
| 悬挑板 | 10% | ≥ 20 个 | GB 50204-2015 附录 E |
| 建筑结构检测 A/B/C 类 | 按表 3.3.10 | — | GB/T 50344-2019 |

梁底面纵向受力钢筋**全部检测**；板类 ≥ 6 根纵向受力钢筋，每根选代表性 3 位置各测 1 点取平均。

### 4. 关键操作技术

#### 4.1 设置参数

- 预设钢筋直径
- 选择量程范围（正确量程可提高精度和最大可测深度）
- 预设钢筋间距

#### 4.2 避开干扰

- 预埋件：管线、铁质材料、金属管
- 钢筋干扰：垂直方向钢筋、斜向钢筋
- 环境干扰：铁质物体堆放处、电线电缆、强磁体（变压器、电焊机）

#### 4.3 相邻钢筋识别

**相邻钢筋净间距 > 保护层厚度时才能有效识别。**

例如：梁底钢筋保护层 35mm，钢筋直径 28mm，轴线间距 60mm
→ 净间距 = 60-28 = 32mm < 35mm → 不能有效识别相邻两根钢筋

#### 4.4 测试操作

| 操作要点 | 原因 |
|---------|------|
| 探头长边**平行**于待测钢筋轴线移动 | 平行时接收信号最强，最接近标准曲线 |
| 接近钢筋轴线正上方时**缓慢移动** | 移动过快导致信号滞后，结果不准 |
| 每根钢筋**连测两次**再换下一根 | 两次差 ≤ ±1mm 有效 |
| 先扫描横向钢筋分布，选**较大空隙**布置竖向测线 | 减少干扰 |

#### 4.5 连续两次测量（重要）

**正确做法**：沿一个方向移至钢筋正上方，读第一次值 → 继续向前几公分 → 反向缓慢移回读第二次值 → 两次值合格后移至下一根。

**错误做法**：沿一条线连续测 6 根读第一次 → 从头再测一次读第二次（两次位置有偏差，可能超过 ±1mm）。

### 5. 测试位置选择

- 结构构件受力最不利的部位
- 有代表性的部位
- 混凝土表面平整、无污物
- 梁体：底面跨中部位
- 顶板：底面中间区域
- 悬挑构件：上表面根部附近

### 6. 结果评定

#### 6.1 允许偏差

| 构件类型 | 允许偏差 (mm) | 1.5 倍限值 |
|---------|-------------|-----------|
| 梁类 | **+10, -7** | +15, -10.5 |
| 板类 | **+8, -5** | +12, -7.5 |
| 墙、柱类 | 无明确验收标准 | — |

#### 6.2 合格判定

| 标准 | 合格条件 |
|------|---------|
| **GB 50204-2015 附录 E** | 合格率 ≥ 90% → 合格；80%~90% → 加倍抽样合并计算；< 80% → 不合格 |
| **GB/T 50344-2019** | 按表 3.5.3-1/3.5.3-2 根据样本容量查表判定 |

#### 6.3 结论模板

```
本次检测的 XX 工程 X 片梁钢筋保护层厚度共计 X 根钢筋 X 个测点，
合格 X 根钢筋 X 个测点，钢筋保护层厚度检验合格率为 XX%。
检测结果中不合格点的最大偏差均不大于规范规定允许偏差的 1.5 倍，
检测结果为合格。

注：多个类型构件均合格/不合格时只在最后写一次；
     多个类型构件有不合格时分开写。
```

### 7. 保护层定义

依据 GB 50010-2010：**钢筋外边缘至混凝土表面的垂直距离**。注意区分设计图纸中的"最外层钢筋保护层"与需检测的"受力钢筋保护层"。

# 第二部分：钢筋锈蚀检测（半电池电位法）

## 检测步骤

### 1. 检测原理

钢筋钝化状态 → 腐蚀电位偏正；活化（锈蚀）状态 → 腐蚀电位偏负。通过铜-硫酸铜参比电极测量钢筋的自然腐蚀电位，判断锈蚀性状。最大电位差可达 500mV。

### 2. 抽样

| 参数 | 要求 | 依据 |
|------|------|------|
| 分类 | 按环境差异及外观检测结果对构件分类 | GB/T 50344-2019 附录 L.0.5 |
| 测区数 | 每类 ≥ 3 个测区 | GB/T 50344-2019 附录 L.0.5 |
| 测区大小 | ≤ 5m × 5m | GB/T 50344-2019 附录 L.0.5 |
| 网格 | 100mm~500mm 方格网 | GB/T 50344-2019 附录 L.0.5 |
| 测点数 | ≥ 20 个（建议 ≥ 30 个） | GB/T 50344-2019 附录 L.0.5 |

### 3. 操作步骤

1. **配制溶液**：过饱和硫酸铜溶液（硫酸铜晶体 + 蒸馏水）
2. **表面处理**：清除涂层、污物，浸湿混凝土表面
3. **剔凿钢筋**：在适当位置剔凿出钢筋，与电压仪负端相连
4. **布置测区**：画网格，交叉点为测点
5. **逐点测量**：半电池依次放在各测点，记录电位值和编号
6. **温度修正**：记录环境温度（22±5℃）
7. **清洁**：及时清除电连接垫表面吸附物

### 4. 判定标准

| 电位水平 (mV) | 锈蚀性状判定 |
|--------------|------------|
| > -200 | 不发生锈蚀的概率 > 90% |
| -200 ~ -350 | 锈蚀性状不确定 |
| < -350 | 发生锈蚀的概率 > 90% |

辅以**局部剔凿钢筋**进行验证。

### 5. 成果表达

- 绘制电位等值线图
- 等值线最大间隔 ≤ 100mV

### 6. 设备维护

| 维护项 | 方法 | 频率 |
|--------|------|------|
| 铜棒清洗 | 稀盐酸轻擦 + 蒸馏水清洗（**禁用钢毛刷**） | 每次使用后 |
| 硫酸铜溶液更换 | 更换后用甘汞电极校准 | 约半年 |

---

## 验证条件

当以下情况之一出现时，应进行**局部剔凿验证**（DB11/T 365-2016 §5.2.7）：

1. 认为相邻钢筋对检测结果有影响
2. 钢筋公称直径未知或有异议
3. 钢筋实际根数、位置与设计有较大偏差

验证点数按最小样本容量（表 5.2.7），合格判定按表 5.2.9。
