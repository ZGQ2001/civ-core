> **适用范围**：混凝土构件抗压强度检测，适用于回弹法有争议、龄期超出回弹曲线适用范围、或需精确评定的场景。
> **涵盖方法**：钻芯法直接检测、回弹法+钻芯修正。

## 一、钻芯法直接检测混凝土强度（检测批）

### 1.1 检测步骤

1. **选点**：按钻芯规 JGJ/T 384 有关规定确定芯样数量和位置
2. **取芯**：使用钻芯机在构件上钻取芯样
3. **试验**：实验室进行抗压试验，记录每个芯样的抗压强度值 `f_cu,cor,i`
4. **计算**：见下方公式
5. **报告**：按 `precision_rules.md`：
   - 芯样强度均值 `f_cu,cor,m` → 精确至 **0.1 MPa**
   - 标准差 `S_cu` → 精确至 **0.01 MPa**
   - 推定值 `f_cu,e` → 精确至 **0.1 MPa**

### 1.2 计算公式（JGJ/T 384-2016 §6.3.2）

检测批的混凝土抗压强度推定值应计算推定区间：

```
推定区间上限值：f_cu,e1 = f_cu,cor,m − k₁ × S_cu
推定区间下限值：f_cu,e2 = f_cu,cor,m − k₂ × S_cu
```

**符号含义**：

| 符号 | 含义 | 精度 |
|------|------|------|
| `f_cu,cor,m` | 芯样试件抗压强度平均值 | 0.1 MPa |
| `f_cu,cor,i` | 单个芯样试件抗压强度值 | 0.1 MPa |
| `f_cu,e1` | 混凝土抗压强度推定上限值 | 0.1 MPa |
| `f_cu,e2` | 混凝土抗压强度推定下限值 | 0.1 MPa |
| `k₁, k₂` | 推定区间上限值系数和下限值系数 | 查附录 A |
| `S_cu` | 芯样试件抗压强度样本的标准差 | 0.01 MPa |

### 1.3 推定系数取值（JGJ/T 384-2016 附录 A）

| 芯样数量 n | k₁（上限系数） | k₂（下限系数） |
|-----------|--------------|--------------|
| … | 查附录 A 表 | 查附录 A 表 |

> **示例**：当 n ≤ 10 时，需按 JGJ/T 384-2016 附录 A 查 k₁/k₂ 系数。如 `k₁=1.11397`, `k₂=2.56600`（模板示例值，实际需查表）。

## 二、回弹法+钻芯修正（检测批）

### 2.1 何时需要钻芯修正

| 条件 | 是否需要钻芯修正 |
|------|----------------|
| 龄期以内 | **不需要**，回弹即可 |
| 龄期以外 | **必须**钻芯修正回弹 |
| 无设计图纸 | **必须**钻芯修正 |
| 砌体结构中混凝土构件 | 可采用龄期修正 |
| 次梁、楼板 | 可在楼梯间、休息平台等部位取芯 |
| 回弹结果有争议 | 钻芯修正验证 |

## 三、检测批推定值选取规则

| 测区数量 | 推定值选取方式 | 依据 |
|---------|--------------|------|
| ≥ 10 测区 | 推定区间上限值（上限-下限差值满足条件时） | GB/T 50784 §4.2.11 |
| < 10 测区 | 取最小值 | — |
| 钻芯法 | 按 §6.3.2 计算推定区间 | JGJ/T 384 |

## 四、常见错误

| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 仅钻芯不修正 | 龄期外仅钻芯验证回弹结果 | 龄期外必须用芯样修正回弹数据 |
| 钻芯系数搞错 | n≤10 时未查附录 A | 按实际芯样数量查表确定 k₁/k₂ |
| 精度错误 | 推定值写 `25.36MPa` | 应为 `25.4MPa`（0.1 MPa） |
| 精度错误 | 标准差写 `3.2MPa` | 应为 `3.20MPa`（0.01 MPa） |
| 修正量遗漏 | 回弹+钻芯只报回弹结果 | 必须附修正量计算表 |
