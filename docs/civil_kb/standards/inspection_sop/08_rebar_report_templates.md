## 一、钢筋保护层厚度检测报告模板

### 1.1 检测依据

依据《钢筋保护层厚度和钢筋直径检测技术规程》DB11/T 365-2016，采用钢筋探测仪。

按 GB/T 50784-2013 §9.3.5 和 §3.4.7 计算均值推定区间：

```
μ_u = m + k_0.5 × s
μ_l = m − k_0.5 × s
```

| 符号 | 含义 |
|------|------|
| `μ_u` | 均值推定区间的上限值 |
| `μ_l` | 均值推定区间的下限值 |
| `m` | 样本均值 |
| `s` | 样本标准差 |

### 1.2 推定值选取规则

| 条件 | 处理方式 |
|------|---------|
| `μ_u − μ_l` ≤ 均值 × 10% | 可按上限值或下限值确定（模板约定取**上限值**） |
| `μ_u − μ_l` > 均值 × 10% | 补充检测或重新划分检测批 |
| 不具备补充条件 | 以**最不利检测值**作为推定值 |

## 四、钢筋力学性能检测（里氏硬度法）

> 此项独立于保护层/间距/直径检测，当需要检测钢筋实际钢材牌号时执行。

### 4.1 检测方法

依据 GB/T 50344-2019 附录 J，采用里氏硬度计检测钢筋表面硬度，换算为抗拉强度特征值。

### 4.2 牌号对照速查

| 钢材牌号 | 抗拉强度标准值 (N/mm²) |
|---------|---------------------|
| Q235 (HRB235) | 370 |
| HRB335 | 455 |
| Q355 | 470 |
| HRB400 | 540 |
| HRB500 | 630 |

### 4.3 报告措辞示例

```
依据《建筑结构检测技术标准》GB/T 50344-2019 中附录 J 的规定，采用里氏硬度计
对构件中的钢筋表面硬度进行检测。
该房屋所抽检带肋钢筋最小抗拉强度特征值为 465N/mm²，接近钢材牌号 Q355 对应
抗拉强度标准值 470N/mm²，钢筋牌号可取 HRB335。
```

## 五、常见报告编写错误

| 错误类型 | 说明 | 正确做法 |
|---------|------|---------|
| 柱墙保护层检测 | 无评价依据仍检测 | 柱墙保护层可以不检测 |
| 保护层精度错误 | 写 `27mm` | 应为 `27.1mm`（1 mm 精度） |
| 钢筋直径精度错误 | 写 `25mm` | 应为 `25.0mm`（0.1 mm 精度） |
| 漏做符合性判定 | 有图不判定 | 有图必须按 GB/T 50784 做计数抽样满足性判定 |
| 箍筋间距漏判 | 只判主筋间距 | 箍筋间距允许偏差 ±20mm 必须一并判定 |
| 钢筋牌号误判 | 力学性能值直接赋牌号 | 取"接近"的牌号，写明判断依据 |
| 有图/无图措辞混用 | 无图用"满足设计" | 无图只描述实测值，不做符合性判定 |
