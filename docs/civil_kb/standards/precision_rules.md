# 数据处理精度规范

> **用途**：各检测项目的数据处理精度（小数保留位数）汇总。编写报告时参照本表确保数据格式规范。

---

## 一、钢结构检测

| 检测项目 | 数据类型 | 精度要求 | 依据 |
|---------|---------|---------|------|
| 里氏硬度 | 测区平均硬度 $HL_m$ | 整数 | GB/T 50344-2019 附录 N |
| 里氏硬度 | 原始测点值 $HL$ | 整数（仪器直接读取） | — |
| 截面尺寸 | 高度/宽度/厚度 | 0.1 mm（游标卡尺） | — |
| 防腐涂层厚度 | 测点值 | 1 μm | GB/T 50621-2010 |
| 防火涂层厚度 | 测点值 | 0.5 mm（超薄/薄型）、1 mm（厚型） | GB/T 50621-2010 |
| 挠度 | 计算值 | 0.1 mm | GB 50017-2017 |
| 垂直度/倾斜率 | 计算值 | 0.01% | GB 50205-2020 |

## 二、混凝土结构检测

| 检测项目 | 数据类型 | 精度要求 | 依据 |

| 混凝土抗压强度（钻芯法） | 芯样强度 $f_{cu,cor}$ | 0.1 MPa | JGJ/T 384-2016 |
| 混凝土抗压强度（钻芯法） | 标准差 $S_{cu}$ | 0.01 MPa | JGJ/T 384-2016 |
| 混凝土抗压强度（钻芯法） | 推定值 $f_{cu,e}$ | 0.1 MPa | JGJ/T 384-2016 |

## 四、常见精度错误汇总

| 错误类型 | 错误示例 | 正确示例 | 涉及规范 |
|---------|---------|---------|---------|
| 回弹值精度错误 | `Rm = 42.35` | `Rm = 42.4` | JGJ/T 23-2011 |
| 碳化深度精度错误 | `dm = 2.3mm` | `dm = 2.5mm`（0.5mm 步进） | JGJ/T 23-2011 |
| 钢筋直径精度错误 | `d = 25mm` | `d = 25.0mm` | GB/T 50784-2013 §9.4.2 |
| 砌体强度精度错误 | `f = 2.3MPa` | `f = 2.30MPa` | GB/T 50315-2011 §15.0.10 |
| 砂浆强度精度错误 | `f2 = 5.23MPa` | `f2 = 5.2MPa` | GB/T 50315-2011 §15.0.10 |
| 钻芯法推定值精度错误 | `f_cu_est = 25.36MPa` | `f_cu_est = 25.4MPa` | JGJ/T 384-2016 |
| 钻芯法标准差精度错误 | `S_cu = 3.2MPa` | `S_cu = 3.20MPa` | JGJ/T 384-2016 |

## 五、精度速查口诀

```
砌体压剪零点几，砂浆保留一位数；
回弹取整零点一，碳化半步零点五；
钢筋直径零点一，钻芯推定也一位；
芯样标准两位记，前后一致才算对。
```
