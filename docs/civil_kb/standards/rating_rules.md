# 鉴定评级规则速查

> **依据**：《房屋结构综合安全性鉴定标准》DB11/T 637-2024（安鉴标）、《既有建筑鉴定与加固通用规范》GB 55021-2021、《民用建筑可靠性鉴定标准》GB 50292-2015。
>
> **用途**：快速查阅构件 → 子系统 → 鉴定系统三级评级的规则、符号、判定条件。

---

## 一、安全性鉴定评级

### 1.1 构件评级（层次一）

| 级别 | 含义 | 判定依据 |
|------|------|---------|
| **au** | 安全 | 构件承载能力、构造连接、位移变形、裂缝损伤均满足规范要求 |
| **bu** | 基本安全 | 略有不足，但不影响构件整体安全性 |
| **cu** | 不安全 | 不满足要求，需采取处理措施 |
| **du** | 极不安全 | 严重不满足要求，存在安全隐患 |

**构件评级路径**（GB 55021-2021 / DB11/T 637-2024）：

```
┌─────────────────────────────────────────────────┐
│   构件安全性 = min(承载能力评级, 构造连接评级,     │
│                    位移变形评级, 裂缝损伤评级)    │
└─────────────────────────────────────────────────┘
```

四个评价维度：

| 维度 | 评价内容 | 关键标准条款 |
|------|---------|------------|
| **承载能力** | 构件抗力/效应比 R/(γ₀S) | DB11/T 637 §4.x |
| **构造与连接** | 连接方式、节点构造、锚固长度 | DB11/T 637 §4.x |
| **位移或变形** | 挠度、垂直度、倾斜 | DB11/T 637 §4.x |
| **裂缝或其他损伤** | 裂缝宽度/深度/分布、材料劣化 | DB11/T 637 §4.x |

### 1.2 构件 vs 构件集

| 概念 | 定义 | 来源 |
|------|------|------|
| **构件** | 单个结构构件 | GB 55021-2021 §4.1.1 |
| **构件集** | 同类构件的集合（主要构件集 / 一般构件集） | GB 50292-2015 表 3.3.1 |

**关键区别**：

- 构件独立评级，不涉及百分比
- 构件集评级含**不合格构件占比**判定：`构件集等级 = f(构件等级分布, 百分比阈值)`
- 构件集**不跨层**划分：同层间类型构件归入同一构件集
- 构件集**不跨类型**：次梁和板均属一般构件集，但类型不同不能合并
- **局部鉴定**按构件集层次评级（构件集与子系统同层）

### 1.3 子系统评级（层次二）

| 级别 | 含义 |
|------|------|
| **Au** | 安全 |
| **Bu** | 基本安全 |
| **Cu** | 不安全 |
| **Du** | 极不安全 |

子系统按承载功能分类：
- 主体结构子系统（地基基础 + 上部承重结构）
- 围护结构子系统

### 1.4 鉴定系统评级（层次三 — 房屋整体安全性）

| 级别 | 含义 |
|------|------|
| **Asu** | 满足安全性要求，不影响整体安全性能 |
| **Bsu** | 基本满足安全性要求 |
| **Csu** | 不满足安全性要求 |
| **Dsu** | 严重不满足安全性要求 |

## 二、抗震鉴定评级

### 2.1 分级

| 级别 | 含义 |
|------|------|
| **Ae** | 抗震能力满足要求 |
| **Be** | 基本满足抗震要求 |
| **Ce** | 不满足抗震要求 |
| **De** | 严重不满足抗震要求 |

### 2.2 依据规范（按建筑类别）

| 建筑类别 | 抗震鉴定依据 |
|---------|------------|
| A/B 类 | GB 50023-2009 抗震鉴定标准 或 DB11/T 637-2024 附录 E |
| C 类 | GB/T 50011-2010 抗震设计规范（2016 版在设计中使用） |

### 2.3 抗震措施检查要点（砌体结构示例）

- 房屋总高度和层数
- 横墙最大间距
- 圈梁布置与构造
- 构造柱设置
- 屋面型式（宜采用轻型屋盖）
- 外墙尽端至门窗洞边最小宽度

### 2.4 鉴定系统抗震评级

| 级别 | 含义 |
|------|------|
| **Ase** | 满足现行抗震要求 |
| **Bse** | 基本满足抗震要求 |
| **Cse** | 不满足抗震要求 |
| **Dse** | 严重不满足抗震要求 |

## 四、特殊构件损伤判定规则

### 4.1 木构件

受压或受弯木构件干缩裂缝满足以下条件时，直接定义为**危险构件**（JGJ 125-2016 §5.5.3）：

- 裂缝深度 > 直径的 **1/2**，且
- 裂缝长度 > 构件长度的 **2/3**

→ 可直接评为 **cu** 或 **du** 构件。

### 4.2 混凝土构件外观缺陷

| 缺陷位置 | 评价方式 |
|---------|---------|
| **构件本体** | 不参与评级，只描述并给出处理意见 |
| **节点处**（如梁柱节点） | 依据严重程度对相关联构件评为 **bu ~ du** |

### 4.3 砌体构件外观缺陷

| 缺陷位置 | 评价方式 |
|---------|---------|
| **构件本体**（风化酥碱、剥落、砂浆粉化） | 不参与评级，只描述并给出处理意见 |
| **节点处**（纵横墙交接咬槎差、钢屋架与墙体仅嵌入未连接） | 对相关联构件评为 **bu ~ du** |

砌体缺陷（砌筑方法、灰缝质量和砌筑偏差）放在**连接和构造评价**中。

### 4.4 单层钢结构宽厚比

| 规范版本 | 宽厚比要求 |
|---------|-----------|
| GBJ 11-89 | 仅对 9 度单层钢结构厂房提出要求 |
| GB 50011-2001 | §9.2.12：单层框架柱、梁截面板件宽厚比限值 |
| GB 50011-2010 | §9.2.14：区分重屋盖/轻屋盖，按性能目标确定 |

> DB11/T 637-2024 对单层钢结构房屋**未提及宽厚比**。但宽厚比是重要评级项，不满足直接 **De**。

## 五、报告编写中鉴定评级常见错误

1. **缺构件集百分比**：构件集评级必须给出不合格构件占比。
2. **构件集跨层**：构件集不要跨层，应按层划分。
3. **构件集跨类型**：次梁和板类型不同不能合并为同一构件集。
4. **评 au 也需四方面汇总**：每个构件都应给出承载能力/构造/位移/裂缝四方面评价。
5. **重点描述非 au 级构件**：au 级构件简写，非 au 级必须详细说明不满足项。
6. **前后不对应**：数据表与评级结论必须一致。
7. **评价依据不充分**：评级必须有明确的规范条款支撑。
