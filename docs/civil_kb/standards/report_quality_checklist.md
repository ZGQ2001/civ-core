# 报告编写质量管控要点

> **用途**：检测鉴定报告编写和审核时的检查清单。覆盖信息页、检测数据、鉴定评级、图纸表述等各环节常见错误。

---

## 一、信息页核对

| 检查项 | 要求 | 常见错误 |
|--------|------|---------|
| **结构类型** | 明确写明（砌体/框架/钢结构/混合） | 遗漏或写错 |
| **建筑类别** | A/B/C 类 | 与抗震鉴定依据不一致 |
| **实际使用性质** | 当前实际用途 | 照抄设计用途 |
| **委托单位** | 写全称，不可简写 | 缩写/错字 |

---

## 二、检测数据

### 2.1 检测照片规范

| 检查项 | 要求 |
|--------|------|
| 剔凿面积 | 必须足够大，露出内部钢筋/钢材（混凝土构件钢筋剔凿不能只剔一点） |
| 缺陷反馈 | 照片中可见缺陷（裂缝、锈蚀等）必须在报告中描述 |
| 两种材料 | 照片中同时出现两种材料（如砖+混凝土）要注明检测对象 |
| 测区编号 | 混凝土构件照片必须有测区编号标记 |

### 2.2 数据处理

| 检查项 | 要求 |
|--------|------|
| 精度 | 按 precision_rules.md 对照检查所有数据的小数位数 |
| 推定值选取 | ≥10 测区 → 推定区间上限值；<10 测区 → 最小值 |
| 符合性判定 | 有设计图纸时需做检验批符合性判定（GB/T 50784-2013 §9.2.8） |
| 回弹抽样数量 | 批量检测 ≥30% 且 ≥10 件（JGJ/T 23-2011 §4.1.3） |
| 钻芯系数 | n≤10 时需按 JGJ/T 384-2016 附录 A 查 k₁/k₂ 系数 |

### 2.3 里氏硬度专项

| 检查项 | 要求 |
|--------|------|
| 测区数量 | ≥3 个/构件（GB/T 50344-2019 附录 N.2.2） |
| 打磨面积 | ≥30mm×60mm（GB/T 50344-2019 附录 N.2.3） |
| 测区覆盖 | 腹板、翼缘均需布置测区 |

### 2.4 涂层厚度专项

| 检查项 | 要求 |
|--------|------|
| 防腐涂层测点 | 5 处 × 3 点，间距 50mm |
| 防火涂层测点 | 梁柱每隔 3m 一截面，≥2 截面；楼板对角线上每米 1 点 |
| 双条件判定 | 每处平均值 ≥85% 设计值 **且** 构件平均值 ≥ 设计值 |

## 三、鉴定评级

| 检查项 | 要求 | 常见错误 |
|--------|------|---------|
| 构件数量 | 报告中的构件数量与检测数据一致 | 漏计或多计 |
| 评级路径 | 承载能力 + 构造连接 + 位移变形 + 裂缝损伤 → 取最低 | 遗漏维度 |
| 非 au 级构件 | 必须详细说明不满足项及规范依据 | 只写"bu"不加解释 |
| 构件集百分比 | 构件集评级必须给出不合格占比 | 缺百分比 |
| 构件集不跨层 | 同层构件才能合并 | 跨层合并 |
| 构件集不跨类型 | 次梁和板不能合并 | 类型混淆 |
| 损伤 vs 缺陷 | 混凝土/砌体本体缺陷不参与评级，只描述；节点缺陷影响关联构件 | 混淆概念 |
| 木构件 | 裂缝深 > 直径 1/2 且长 > 构件 2/3 → 危险构件 cu/du | 漏判 |
| 单层钢结构宽厚比 | 不满足 → 直接 De | 遗漏此项 |

---

## 四、图纸/平面图

| 检查项 | 要求 |
|--------|------|
| 梁尺寸 | 图纸上必须标注各梁截面尺寸 |
| 预制板方向 | 平面图中必须标明预制板铺设方向 |
| 屋面板方向 | 坡屋面必须标明屋面板铺设方向 |
| 屋架截面尺寸 | 钢木屋架必须标注截面尺寸 |
| 构造柱 | 平面图中必须标注构造柱位置 |
| 坡屋面高度 | 按**檐口高度**和**屋脊高度**分别标注（GB 50352-2019 §2.0.13） |

---

## 五、计算分析

| 检查项 | 要求 | 常见错误 |
|--------|------|---------|
| 荷载取值 | 填充墙荷载需注明取值依据 | 直接取值不注明 |
| 板边界条件 | 明确板的支座条件（简支/固结/连续） | 边界条件与实际情况不符 |
| 檩条连续性 | 连续檩条搭接长度 ≥ 10% 跨度，嵌套部分螺栓连接（GB 51022-2015 §9.1.10） | 忽略搭接松动引起的刚度折减 |
| 檩条侧向支撑 | 直立缝锁边/扣合式屋面板不能作为檩条侧向支撑（GB 51022-2015 §11.1.6） | 错误认为屋面板可作支撑 |
| 隅撑作用 | 隅撑不能作为梁的固定侧向支撑，仅为弹性支座（GB 51022-2015 §7.1.6 条文说明） | 默认取两倍檩距为计算长度 |
| 分项系数 | 原结构安全性鉴定按原建造规范；加固/改变用途按现行规范（GB 55021-2021 §4.2.2） | 混淆两类鉴定目的 |

---

## 六、前后对应性检查

| 检查项 | 说明 |
|--------|------|
| 数据表 ↔ 评级结论 | 检测数据中的所有异常值必须在评级中体现 |
| 平面图 ↔ 文字描述 | 图纸标注的构件尺寸/数量与文字完全一致 |
| 信息页 ↔ 正文 | 结构类型/建筑类别在信息页和正文中保持一致 |
| 结论页 ↔ 评级细节 | 结论不能出现正文中未提及的评定 |

## 七、结论页格式

- 结论页使用**建委官网范本**：https://www.beijing.gov.cn/zhengce/zhengcefagui/qtwj/201107/t20110712_566930.html
- 结论措辞与评级等级严格对应，不能模糊化
- 非 au/Au 级必须附处理建议
