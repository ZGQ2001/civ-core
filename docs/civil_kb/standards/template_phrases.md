## 二、鉴定结论

### 2.1 标准结论模板

```
依据《房屋结构综合安全性鉴定标准》DB11/T 637-2024相关规定，对XX项目进行
综合安全性鉴定，结论如下：
（1）房屋结构安全性评级为Asu级；
（2）房屋建筑抗震鉴定评级为Ase级；
（3）房屋结构综合安全性评级为Aeu级。
```

### 2.2 评级解释

```
Asu级：满足《房屋结构综合安全性鉴定标准》DB11/T 637-2024的安全性要求，
      不影响整体安全性能；
Ase级：满足现行国家标准《建筑抗震鉴定标准》GB50023和《房屋结构综合安全性
       鉴定标准》DB11/T 637-2024的抗震要求，不影响整体抗震性能；
Aeu级：房屋结构安全性满足《房屋结构综合安全性鉴定标准》DB11/T 637-2024的
       安全性要求，具有足够的承载能力和结构整体稳定性；建筑抗震能力满足现行
       国家标准《建筑抗震鉴定标准》GB 50023和《房屋结构综合安全性鉴定标准》
       DB11/T 637-2024的要求，在预期的后续工作年限内不影响整体承载和抗震性能。
```

### 2.3 处理建议

```
（1）对该建筑进行定期检查和维护，发现问题及时处理；
（2）未经技术鉴定或设计许可，不得随意改变房屋使用功能、增加使用荷载。
```
